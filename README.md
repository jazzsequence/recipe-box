## Description ##

Easily store and publish recipes in WordPress.

## Installation ##

### Manual Installation ###

1. Upload the entire `/recipe-box` directory to the `/wp-content/plugins/` directory.
2. Activate Recipe Box through the 'Plugins' menu in WordPress.
